Asteroids

Startup:
- To run the game select the top menu "Game/Start Game" or find a scene called "Main", open it and press the Play button.
- In case you need to reset Best Score select in the top menu "Game/Delete Game data".

Setup:
- All settings and configurations files can be found in the Resources folder.
- GameConfiguration allows modifying general settings such as LivesCount, DelayBetweenSessions etc.
- ShipSettings allows changing Ship-related settings such as speed, shot delay etc.
- AsteroidsConfiguration allows adding/changing asteroids settings for specific asteroid type.
- "Main" scene contains the object AsteroidsSpawner that allows adjusting spawn probability for each spawner in the scene.
